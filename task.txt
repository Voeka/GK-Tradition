1. Создать файл /index.php
    В файле сверстать форму авторизации (ссылается на себя) с использованием Bootstrap 5.
     Список полей в форме: "Е-майл", "Пароль" и кнопка.
     1. В этом файле написать прослушивание POST значений, если в них есть логин и пароль, провести их проверку:
         - Почту надо проверить на корректность и её значение должно быть проверка@tradicia-k.ru
         - Пароль равен `pass`
     2. Если е-майл и пароль проходят проверку, должен сработать 302 редирект на /form.php

1. Создать файл /form.php
    В файле сверстать форму на Bootstrap 5, которая ссылается на /save.php.
    Список полей в форме:
    1. Поле "Наименование" (однострочное);
    2. Поле "Категория" (select элемент со списком значений: Default, Gidromolot, kovshi, vibropogruzhateli, mulchery) с поиском по списку;
    3. Поле "Дата" (в формате DD-MM-YYYY) с плагином календаря (на ваш выбор);
    4. Поле "Описание" с множеством строк (textarea);
    5. Поле "Активное" (checkbox);
    6. Кнопки Отправить и Отмена (сбрасывает поля формы от указанных значений);

3. Создать файл /save.php
    Реализовать проверку полей:
    1. "Наименование" на пустоту;
    2. "Категория" на наличие значения из допустимого списка;
    3. Дата на корректность формата DD-MM-YYYY;
    4. В поле описание надо предусмотреть парсер получения всех гиперссылок указанных там в формате html;
    Полученные и проверенные значения записать в сессию что-бы при повторном открытии /form.php они подставились в поля.
    В добавок в конце полученные значения преобразовать в JSON и вывести на экран через echo, согласно типу JSON;

4. Разработать тесты написанного кода (по вашему смотрению);

Стиль написания PSR-1, PSR-2, PSR-12.
Синтаксис PHP >= 8.0 и declare(strict_types=1).

Работу можно выгрузить на https://phpsandbox.io/
